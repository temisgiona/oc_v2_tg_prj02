Notice utilisation : 
Préalable :
Vérifiez la présence des fichiers :
	main.py, 
	file_creation.py  
	booktoscrap.py
Vérifier la présence des répertoires "data" et "image" à la racine du projet.
Les fichiers csv seront stockés dans le repertoires "data" et les images des livres scrapés dans le repertoire "image".

Pour faire fonctionner le programme :  executer main.py.


Instructions for use : 
Prerequisite:
Check the presence of the files :
	main.py, 
	file_creation.py  
	booktoscrap.py
Check the presence of the "data" and "image" directories at the root of the project.
The csv files will be stored in the "data" directory and the images of the scrap books in the "image" directory.

To run the program : execute main.py.
